# Gambler

Godot 4.3 卡牌对战游戏

## 快速开始

1. 用 Godot 4.3 打开项目
2. 运行 `MainV2.tscn` 测试最新版本
3. 查看 [docs/index.md](docs/index.md) 了解文档结构
